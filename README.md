## About Project

This project is implementation of machine learning to classified lung disease cancer. There are 4 categories lung opacity, normal, and viral pneumonia

--- 
## Built With
- Python 3.10+
- Jupyter Notebook
- torch
- torchvision
- matplotlib
- Anaconda for Python Distribution
  [Download Anaconda](https://www.anaconda.com/download)
## Setup and Installation
### 1. Clone Repository

```bash id="6ncgzj"
[git clone https://github.com/IsadadJhn/Brain_Tumor_Classifier.git]
cd Brain_Image_Classifier
```
### 2. Dataset
This project is uses a dataset from kaggle
[Download Here](https://www.kaggle.com/datasets/masoudnickparvar/brain-tumor-mri-dataset)

### 3. Install Dependency
```bash id="6ncgzj"
pip/pip3 install torch torchvision matplotlib jupyterlab 
```
